#### 기존 126 버젼 삭제
```less
# 크로미움 버젼 확인 (부산문은 120버젼)
chromium-browser --version
Chromium 126.0.6478.164 Built on Debian , running on Debian 11

# 기존 크로미움 삭제
sudo apt remove chromium chromium-browser chromium-driver
sudo apt autoremove

sudo apt purge chromium chromium-browser chromium-driver
sudo apt purge chromium chromium-browser chromium-chromedriver
sudo apt autoremove
sudo apt update
pip cache purge

# 잔여 설정 파일 정리
sudo apt purge rpi-chromium-mods
sudo apt autoremove
```

#### sh 쉘 등록해서 크론탭에서 이거 실행
```less
# .py로 크론탭에서 실행하면 오류 가능성있음
sudo nano run_updateEco.sh

#!/bin/bash
source /home/pi/.virtualenvs/elcsoft/bin/activate
python /home/pi/www/Eco/updateEcoData.py >> /home/pi/www/Eco/cron.log 2>&1

sudo chmod +x /home/pi/www/Eco/run_updateEco.sh
```

#### 크론탭 등록
```less
crontab -e

0 * * * * /home/pi/www/Eco/run_updateEco.sh
```
